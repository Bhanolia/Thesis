# **manager**

this app is required for thesis. The manager is built using the Flutter framework and firebase. The manager application has functions to help manage archive data and announcements.git 

## This application has the following features:
- integrated with firebase

- has 2 user classes admin and user.

- for admins can create, modify and delete archive data and announcements.

- displays notifications when there is a new announcement.

- automatically archive 30 days old announcements.

- can download announcement attachments and archives.
